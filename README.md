# smolmoji

Easily create and edit pixel art emojis or icons with AI

![smolmoji - pixel art emoji creator](public/smolmoji-overview.jpg)

![smolmoji example](public/smolmoji-example.gif)

**Live:** [smolmoji.com](https://smolmoji.com)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
git clone <repo-url>
cd smolmoji
npm i
npm run dev
```
