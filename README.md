# smolmoji

Easily create and edit pixel art emojis or icons with AI

<img src="public/smolmoji-overview.jpg" alt="smolmoji - pixel art emoji creator" width="75%" />

**Live:** [smolmoji.com](https://smolmoji.com)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
git clone <repo-url>
cd smolmoji
npm i
npm run dev
```
