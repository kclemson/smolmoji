# smolmoji

Create pixel art emojis with AI ✨

**Live:** [smolmoji.com](https://smolmoji.com)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
git clone <repo-url>
cd smolmoji
npm i
npm run dev
```
